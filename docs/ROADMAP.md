# ROADMAP.md

## Vision

Un appartement aménagé sur mesure, conçu sur maquette 3D fidèle, dont une grande partie du
mobilier est autoconstruite à partir de plans de fabrication précis.

## Phase 1 — Coque 3D (en cours)

- [x] Plan PDF → DXF à l'échelle réelle
- [x] Murs 3D (composants, hauteur 2500 mm)
- [x] Import dans SketchUp (`Plan 3D.skp`)
- [ ] Portes + fenêtres en vraies ouvertures (+ sens d'ouverture)
- [ ] Reporter contraintes techniques (eau, élec, gaine, radiateurs, faux-plafonds)
- [ ] Confirmer hauteur sous plafond

## Phase 2 — Zoning & circulation

- [ ] Définir usages par pièce + besoins (couchages, bureau, rangements, terrasse)
- [ ] Tracer les flux de circulation (passages ≥ 80–90 cm)
- [ ] Grands partis d'aménagement par pièce (2-3 variantes en scènes SketchUp)

## Phase 3 — Agencement détaillé (pièce pilote)

- [ ] Choisir la pièce pilote (cuisine ou entrée/dressing recommandé)
- [ ] Implantation détaillée validée
- [ ] Lister le mobilier sur mesure à fabriquer

## Phase 4 — Conception mobilier + fabrication

- [ ] Modéliser chaque meuble en composant, à l'épaisseur réelle des matériaux
- [ ] Installer/configurer OpenCutList
- [ ] Générer fiches de débit + calepinage panneaux + estimatif
- [ ] **Relevés réels sur site** → figer les cotes de fabrication
- [ ] Fabrication échelonnée pièce par pièce

## Idées futures

- Bibliothèque de composants meubles réutilisables.
- Pièces de jonction / gabarits via laser ou impression 3D.
- Rendu réaliste pour visualisation (matériaux, lumière).

## Critères avant d'ajouter une pièce/un meuble au périmètre

- Besoin réel et usage clair.
- Fabricable avec les outils et compétences disponibles.
- Transportable/montable à l'étage R+7.
- Compatible avec les contraintes techniques figées.
